## 💡主要功能

1. 支持为段落打**标签**
2. 支持设置**搜索域**（设置搜索**数据库表的字段**）
3. 支持使用**正则表达式搜索**
4. 支持**导入导出数据库**为文件
5. 支持**使用 WebDAV 同步**数据
6. 支持根据关键词**自动填充**输入框
7. 支持通过选中段落后的**上下文菜单快捷添加段落**
8. 支持**深色模式**
9. ......
